# NaiwaProxy 使用说明

本文档只说明当前界面中与 **系统代理**、**TUN**、**路由规则** 相关的配置含义。

## 节点管理

节点管理页用于选择当前节点、启用代理，并配置系统代理、TUN 和路由规则。

### 系统代理

系统代理控制 Windows 当前用户的 Internet 代理设置。它不会单独启动 Core，需配合右上角 **启用代理** 使用。

默认端口：

| 类型 | 地址 |
|------|------|
| HTTP | `127.0.0.1:10809` |
| SOCKS | `127.0.0.1:10808` |

选项说明：

| 选项 | 含义 |
|------|------|
| 清除系统代理 | 关闭 Windows 系统代理，并清理 `ProxyServer` 和 `AutoConfigURL` |
| 自动配置系统代理 | Core 运行时，将 HTTP 代理设置为 `127.0.0.1:10809` |
| 不改变系统代理 | 不修改 Windows 系统代理配置 |
| PAC 模式 | 生成 `%APPDATA%\NaiwaProxy\proxy.pac`，本地/局域网地址直连，其余走 HTTP 代理 |

关闭代理或退出应用时，如果当前模式是 **自动配置系统代理** 或 **PAC 模式**，程序会自动清除系统代理。

### TUN

TUN 通过 `sing-box` 创建虚拟网卡，将系统流量转发到本地 SOCKS 端口。

依赖文件位于安装目录 `cores`：

| 文件 | 作用 |
|------|------|
| `wintun.dll` | Windows TUN 驱动 |
| `sing-box.exe` | TUN 运行时 |
| `libcronet.dll` | sing-box 随包依赖 |

使用条件：

| 条件 | 说明 |
|------|------|
| 管理员权限 | 点击 TUN 开关时，如果当前不是管理员，程序会请求 UAC 并以管理员身份重启 |
| 先启用代理 | TUN 会转发到本地 SOCKS，Core 必须先运行 |
| 自动停止 | 关闭代理或退出程序时会停止 TUN |

状态说明：

| 状态 | 含义 |
|------|------|
| 已关闭 | TUN 未启用 |
| 待启动 | 已保存 TUN 开启状态，但当前运行时未启动 |
| 运行中 | sing-box TUN 已启动 |
| 需要管理员 | 当前进程没有管理员权限 |
| 缺少运行时 | `sing-box.exe` 或相关依赖缺失 |

### 路由规则

路由规则决定流量走代理、直连或阻止。修改路由规则后，如果 Core 正在运行，程序会自动重启 Core 使配置生效。

内置模式：

| 模式 | 含义 |
|------|------|
| 全局代理 | 默认全部流量走代理 |
| V4-绕过大陆 (Whitelist) | 私有 IP、`geoip:cn`、`geosite:cn` 直连，其余走代理 |
| 绕过局域网 | 私有 IP、本地网段直连，其余走代理 |
| 直连模式 | 所有流量直连，相当于不代理 |
| 自定义规则 | 使用 **编辑规则** 弹窗中的自定义列表 |

点击 **编辑规则** 可配置自定义路由。每行一条规则。

| 列表 | 出站 |
|------|------|
| 走代理域名 | proxy |
| 直连域名 | direct |
| 阻止域名 | block |
| 走代理 IP / CIDR | proxy |
| 直连 IP / CIDR | direct |
| 阻止 IP / CIDR | block |

匹配优先级：

1. 阻止
2. 直连
3. 代理

域名写法：

| 写法 | 含义 |
|------|------|
| `example.com` | 自动转换为 `domain:example.com` |
| `domain:example.com` | 匹配域名及子域 |
| `full:example.com` | 完整域名匹配 |
| `regexp:^.*\.example\.com$` | 正则匹配 |
| `geosite:category-ads-all` | Xray geosite 列表 |

IP 写法：

| 写法 | 含义 |
|------|------|
| `1.2.3.4` | 单个 IP |
| `1.2.3.0/24` | CIDR 网段 |
| `geoip:cn` | Xray geoip 列表 |

保存自定义规则后，路由模式会自动切换为 **自定义规则**。

## 订阅管理

订阅管理页当前复用节点导入能力。导入后的节点同样受 **系统代理**、**TUN**、**路由规则** 影响。

订阅导入本身不改变系统代理、TUN 或路由规则；只有在节点管理页启用代理后，上述配置才会作用到实际流量。
