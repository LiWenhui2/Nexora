# NaiwaProxy 功能扩展菜单规划

本文档用于规划 NaiwaProxy 后续可扩展能力。当前项目已经具备 VMess 导入、节点保存、Xray Core 启停、系统 HTTP 代理开关、延迟测试、安装包构建等基础能力。后续可以围绕“节点管理、代理模式、订阅、规则、系统集成、可观测性、安全性、用户体验、安装更新”等方向扩展。

## 1. 首页 / 仪表盘

### 1.1 连接状态

- 当前运行状态：未启动、运行中、启动失败、正在重启。
